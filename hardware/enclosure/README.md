# ESP32-S3 Print Bridge 外壳设计

本项目为 ESP32-S3 打印机 Wi-Fi 桥接器提供了一套定制化的 3D 打印外壳方案。该设计主打**极简隐蔽式**风格，旨在让设备能够稳定、安全且美观地贴附在 HP M1136 打印机背部。

## 设计理念：铝合金底壳 + 非金属透波顶盖

由于项目使用的是板载 PCB 天线的 ESP32-S3-N16R8 开发板，如果采用全金属外壳会产生严重的“法拉第笼”效应，导致 Wi-Fi 信号被完全屏蔽。因此，我们采用了一种混合材质的折中与优化方案：

*   **主壳体（底壳及四周）**：采用铝合金 3D 打印（如 AlSi10Mg）。底部保留大平面用于粘贴 3M VHB 双面胶。铝合金材质不仅提供了极佳的散热性能和坚固的物理保护，其表面处理还带来了高级的工业质感。同时，四角和边缘均设计了圆角与微倒角，以防止日常使用时割伤手或刮花打印机。
*   **顶盖板**：采用黑色半透明亚克力或 PC 等非金属透波材质。这使得 Wi-Fi 信号能够畅通无阻，同时由于其半透明特性，用户可以隐约观察到内部 ESP32 状态指示灯的闪烁情况。顶盖通过沉头螺丝与主壳体固定。

## 尺寸适配与测绘

市面上有多种版型的 ESP32-S3 开发板，目前的 3D 模型主要基于最常见的双 Type-C 并在同侧的版型（类似 YD-ESP32-S3）进行初步建模参数设定。

> ⚠️ **非常重要**：在直接导出打样前，务必使用游标卡尺核实您手头实物的具体尺寸，并在生成脚本中进行参数微调。

供参考的通用参数如下：
*   **PCB 尺寸**：约 53.0 mm (长) x 28.0 mm (宽)
*   **PCB 厚度**：1.6 mm
*   **最大整体厚度**：约 6.0 mm（包含底部引脚或焊点。如果去除排针可以做得更薄）
*   **Type-C 接口间距**：位于短边，两个接口的中心距需实测确认。
*   **安装定位孔**：通常为 4 个 M2 或 M2.5 孔。

## Blender 自动化建模与导出指南

为了应对金属 3D 打印对模型公差的极高要求，并提供直观的材质预览，我们采用了基于 **Blender 4.2+** 的 Python 自动化建模脚本。您无需具备任何手动 3D 建模基础，只需输入卡尺测量的参数，即可一键生成参数化、拓扑干净的三维模型。

### 快速生成步骤：

1. **准备环境**：打开 **Blender 4.2** 或更高版本软件。
2. **进入脚本空间**：在顶部菜单栏点击 **Scripting** 标签页，切换到脚本工作区。
3. **载入脚本**：在屏幕下方的文本编辑器区域点击 **"New" (新建)**，然后将本项目 `hardware/enclosure/create_enclosure_in_blender.py` 文件内的全部 Python 代码复制粘贴进去。
4. **修改尺寸参数**：在代码文件头部找到参数定义区，将 `pcb_length`, `pcb_width`, `port_spacing` 等变量修改为您实际测量的数值。代码内部采用公制，输入的数值单位默认为**毫米 (mm)**。
5. **一键生成**：点击文本编辑器菜单栏右上角的 **"Run Script" (▶ 播放图标按钮)**。
6. **预览与导出**：回到 **Layout** 工作区，您将在 3D 视图中看到精确生成的**铝合金底壳**和**亚克力顶盖**（如果未自动切换，可手动调整至 Material Preview 模式以查看真实的金属和玻璃质感）。
7. **交付打样**：确认无误后，在场景中选中生成的底壳对象（即 `Aluminum_Base`），依次点击左上角菜单 `File` -> `Export` -> `Stl (.stl)`，并在弹出的导出设置中勾选 **"Selection Only" (仅导出选中项)**，即可得到用于厂家打样的 STL 模型文件。
